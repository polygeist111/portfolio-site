Archetyp Catalog Generator 2023-2024

Made for Archetyp Alpine Wines, this program dynamically generates a downloadable vector PDF of Archetyp's wine offerings and prices for trade parters, as well as general information and winemaker profiles, with data updated live. If a wine sells out, the generator will omit it. If a new wine is pushed to the online store, the generator will include it immediately. This project is built with p5.js, as well as Zeno Zeng's p5.js-svg and p5.js-pdf tools. Wine data is pulled from Commerce7's REST APIs, and price data is accessed with Google Auth. For the purposes of this demo, Google Auth has been removed and a snapshot of price data from 7/24 has been taken, so that non-employees can still test the system. This is an ongoing project, and future updates to the full program will also occassionally be ported to this demo.

Included with permission from Archetyp Alpine Wines

Forgive the long-winded title, it's a matter of compliance. This program, made in p5.js and purely as a personal project, is a tool to help in creating mosaics with LEGO速 pieces. By uploading your image, panel dimensions (think of it like an aspect ratio), title, and colors (this demo has preset colors, for ease of use), the program will subdivide the image into pieces, and for each piece identify its nearest matching color among available LEGO速 parts. It will then assemble these into all of the graphics needed to create an instruction booklet for your mosaic. This project is a work in progress, and more advanced (but not web-compatible) versions exist that I plan to port over. Come back soon for a more complete version!
